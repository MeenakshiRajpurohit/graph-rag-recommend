# graph-rag-recommend
A Graph-RAG recommendation framework combining Knowledge Graphs, GNNs, and LLM re-ranking to deliver explainable item recommendations on Amazon Reviews 2023. Features node2vec pattern mining, LightGCN/GraphSAGE embeddings, popularity-selective KG retrieval, and category-specific LoRA fine-tuning with LLaMA-2, LLaMA-3, and Qwen2.
